This folder contains a set of notebooks to help users understand how to use this software.

The most natural order of notebooks to follow is:
 - `example.ipynb`: Getting started with the software.
 - `advanced_methods.ipynb`: Going beyond the standard use case of the software.
 - `example_grid_patcher.ipynb`: Fixing interpolation issues due to the gridding schemes of some CMIP6 models. Explaining the issue and the fix.
